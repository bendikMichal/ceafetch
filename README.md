
## ceafetch
- A cross-platform fetch for Windows and Linux

#### Example

```
         ██████╗███████╗ █████╗          USER:  user
        ██ ════ ██ ════╝██ ══██╗         SHELL: /bin/bash 
        ██║     █████╗  ███████║         TERM:  xterm 
        ██║     ██ ══╝  ██ ══██║         LANG:  en_US.UTF-8 
        ╚██████ ███████ ██║  ██║         OS:    Linux 
                                         RAM:   x.xx gb / x.xx gb 
```


### How to build
```sh
make
```
- to test run
```sh
make run
```

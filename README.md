
## ceafetch
- A cross-platform fetch for Windows and Linux

#### Example
![Example](https://github.com/bendikMichal/ceafetch/blob/assets/ceafetch.png)

### How to build
- <strong>make sure you have:</strong>
  * git installed and in PATH
  * internet connection
```sh
make
```
- to test run
```sh
make run

```
